# VP-004: Wikilink resolution correctness

## Property Statement

Given a wiki/index.md containing a known set of page slugs S, and a wiki page body containing wikilinks W:
- All `[[slug]]` references where slug ∈ S → allow (exit 0)
- Any `[[slug]]` reference where slug ∉ S → block (exit 2, E-WIKI-001)
- The check is O(n) in |S| (index-first, not O(n²)) — verified by timing assertion at |S| = 10K

## Verification Mechanism

bats (wiki.bats and hooks.bats) — two layers:

**Layer 1 — pure unit test on resolution algorithm:**
```bash
@test "wikilink-resolve: slug in index → present" {
  local index_fixture="${BATS_TEST_DIRNAME}/fixtures/wiki-index-100.txt"  # 100 slugs
  result=$(bash "${CLAUDE_PLUGIN_ROOT}/hooks/lib/wikilink-resolve.sh" \
    --index "$index_fixture" --slug "zettelkasten")
  assert_equal "$result" "present"
}

@test "wikilink-resolve: slug not in index → absent" {
  result=$(bash "${CLAUDE_PLUGIN_ROOT}/hooks/lib/wikilink-resolve.sh" \
    --index "$index_fixture" --slug "nonexistent-slug-xyz")
  assert_equal "$result" "absent"
}
```

**Layer 2 — hook integration test:**
```bash
@test "validate-wikilink-integrity.sh: broken wikilink → E-WIKI-001" {
  local payload='{"tool":"Write","input":{"path":"wiki/concepts/test.md","content":"...[[broken-link]]..."},"output":{}}'
  echo "$payload" | "${CLAUDE_PLUGIN_ROOT}/hooks/validate-wikilink-integrity.sh"
  assert_failure 2
  assert_output --partial '"code":"E-WIKI-001"'
}
```

**Scale timing assertion (BC-2.05.002):**
```bash
@test "validate-wikilink-integrity.sh: O(n) at 10K pages (< 100ms)" {
  # Generate wiki-index-10k.txt fixture (10K slugs)
  local start; start=$(date +%s%3N)
  echo "$payload_with_valid_link" | "${CLAUDE_PLUGIN_ROOT}/hooks/validate-wikilink-integrity.sh"
  local end; end=$(date +%s%3N)
  local elapsed=$((end - start))
  assert [ "$elapsed" -lt 100 ]
}
```

## Assumed Prerequisites

- `tests/fixtures/wiki-index-100.txt` and `tests/fixtures/wiki-index-10k.txt` generated by `scripts/gen-test-corpus.sh --format wiki-index-only`

## Counterexamples

- A valid wikilink `[[zettelkasten]]` that exists in index.md is blocked (false positive)
- A broken wikilink `[[nonexistent]]` is allowed (false negative — violates BC-2.04.003)
- Resolution time grows super-linearly with index size (O(n²) — violates BC-2.05.002)
